distinct intent, or they cannibalize. Check CONTENT_INDEX before writing.

   "why AI video lacks realism", average-render-to-pro. Heavy coverage.

## Keyword discipline

- One primary keyword + one search intent per article.
- Before writing: grep the repo for the keyword, scan CONTENT_INDEX, scan this
  file. If overlap, narrow the angle, change the intent, or update the existing
  article and add internal links instead of publishing a near-duplicate.
- Record each new article's primary + secondary keywords in CONTENT_LEDGER.md.
