# Runway lance Agent Skills : créer une pub vidéo IA en une commande

Le 2 juillet 2026, Runway a silencieusement poussé une mise à jour qui change pas mal de choses pour les créateurs qui font de la pub : **Agent Skills**. Disponible sur tous les plans, cette fonctionnalité vous permet de déléguer des tâches complexes à un agent IA, directement dans l'interface Runway.

Une seule phrase suffit. "Crée un spot commercial de 15 secondes pour ce produit." L'agent s'en charge.

## Ce que fait concrètement Agent Skills

Agent Skills n'est pas une nouvelle fonction de génération vidéo. C'est une couche d'automatisation par-dessus les outils Runway existants. L'idée : vous donnez un objectif, l'agent enchaîne les étapes pour l'atteindre.

Trois cas d'usage ont été mis en avant dans le changelog officiel :

- **Créer une campagne pub** : l'agent construit plusieurs variantes vidéo pour un produit ou un brief donné
- **Produire un spot commercial** : génération bout en bout, de la première image au montage final
- **Localiser des annonces** : adapter un spot existant à plusieurs langues ou marchés, automatiquement

Ce dernier point est sous-estimé. Localiser une vidéo IA prenait auparavant plusieurs allers-retours entre les outils de voice cloning, de lip sync et de re-génération de plan. Là, ça se fait en une commande.

## Pourquoi c'est un tournant pour les créateurs pub

Jusqu'ici, Runway était fort sur la génération de plan à plan : vous donnez une image, vous obtenez un clip de 4 à 10 secondes. Très bien pour les B-rolls, les transitions, les aperçus. Moins pratique quand votre client veut un spot complet avec cohérence visuelle sur 30 secondes.

Agent Skills déplace le curseur vers la **production agentique**. Vous ne générez plus clip par clip. Vous donnez un brief, l'agent orchestre les ressources.

Le terme "agentic" est partout en 2026, souvent pour rien. Ici, c'est concret : l'agent peut accéder au modèle Runway Characters (déployé en mars 2026 pour verrouiller l'identité visuelle des personnages), combiner les plans générés, et sortir un résultat cohérent.

> 💡 **Le cut de Frank :** Ce qui m'intéresse vraiment ici, c'est la localisation automatique. Pour les créateurs qui travaillent sur des campagnes multi-marchés, c'est des heures gagnées. Reste à tester si la qualité tient sur des accents régionaux difficiles.

## Ce qui change (et ce qui ne change pas) dans votre workflow

**Ce qui change :**
- Les briefs complexes passent maintenant en une étape
- La localisation vidéo devient accesssible sans pipeline personnalisé
- Les itérations client vont plus vite : vous pouvez générer plusieurs versions d'un même spot très rapidement

**Ce qui ne change pas :**
- Runway reste facturable à la seconde de vidéo générée, les coûts peuvent grimper vite sur un brief ambitieux
- La cohérence de personnage dépend toujours de vos références source, l'agent ne peut pas inventer un visage cohérent sans matière première
- La qualité des prompts compte toujours : un brief vague donne un résultat vague, même avec un agent

## Comment accéder à Agent Skills

La fonctionnalité est disponible dès maintenant sur **tous les plans Runway**, y compris les plans d'entrée de gamme. Elle est accessible depuis `app.runwayml.com/agent`.

Runway n'a pas encore publié de documentation détaillée sur les limites par plan (nombre de crédits consommés par tâche agentique, résolution de sortie par défaut). À vérifier avant de lancer une grosse campagne.

## Ce que ça dit sur la direction de Runway

Depuis le partenariat avec Adobe annoncé en janvier 2026 (Gen-4.5 intégré à Firefly), Runway ne se positionne plus comme un outil de niche pour les VFX artists. L'objectif est clairement de capturer le marché de la production pub et du contenu de marque.

Agent Skills s'inscrit dans cette logique. Les annonceurs et les agences n'ont pas le temps de maîtriser les paramètres techniques. Ils veulent donner un brief et obtenir un livrable. C'est exactement ce que promet cette update.

Le vrai test viendra quand des agences utiliseront Agent Skills sur de vrais budgets clients. Les résultats qu'on verra dans les prochaines semaines diront si la promesse est tenue ou si l'agent produit encore des artefacts qui nécessitent une heure de correction en post.

## Ce que vous devriez faire maintenant

Si vous faites de la pub ou de la production de contenu de marque, la première chose à faire est de tester Agent Skills sur un brief simple : un produit physique que vous avez déjà en référence, un ton clair, un format court (15s max).

Observez ce que l'agent produit sans instruction détaillée, puis affinez. C'est la meilleure façon de comprendre où ses limites se situent avant de le proposer à un client.

Les créateurs qui se forment à écrire des briefs clairs pour les agents IA auront un avantage réel dans les 18 prochains mois. Ce n'est plus une compétence optionnelle.

## FAQ

### Qu'est-ce que Runway Agent Skills exactement ?
Runway Agent Skills est une couche d'automatisation agentique intégrée à Runway depuis le 2 juillet 2026. Vous donnez un objectif en langage naturel (créer une campagne, produire un spot, localiser des annonces) et l'agent orchestre les outils Runway pour atteindre ce résultat. Il est disponible sur tous les plans depuis `app.runwayml.com/agent`.

### Sur quels plans Runway est disponible Agent Skills ?
Agent Skills est disponible sur tous les plans Runway, y compris les plans d'entrée de gamme. Runway n'a pas encore publié de détail précis sur la consommation de crédits par tâche agentique. Vérifiez votre quota avant de lancer une production longue.

### Peut-on localiser automatiquement une vidéo IA avec Runway Agent Skills ?
Oui, la localisation fait partie des trois cas d'usage officiels annoncés par Runway. L'agent peut adapter un spot existant à plusieurs langues ou marchés. La qualité dépend de la langue cible et des ressources vocales disponibles. Pour les langues à accent régional marqué, un contrôle manuel reste conseillé.

### Agent Skills remplace-t-il la génération clip par clip dans Runway ?
Non, Agent Skills s'ajoute aux outils existants. Vous pouvez toujours générer des clips manuellement, ajuster les paramètres, utiliser les modes texte-to-video ou image-to-video comme avant. Agent Skills est une couche supplémentaire pour déléguer les tâches complexes et multi-étapes.

### Runway Gen-4.5 est-il utilisé par Agent Skills ?
Runway utilise ses modèles internes pour les générations dans Agent Skills, dont Gen-4.5 et Runway Characters pour la cohérence des personnages. La documentation technique complète n'est pas encore publiée, mais les fonctionnalités disponibles dans votre plan s'appliquent aussi dans le mode agentique.

### Comment écrire un bon brief pour Runway Agent Skills ?
Donnez un contexte précis : le produit ou le sujet, le ton souhaité (cinématique, dynamique, épuré), le format de sortie (9:16, 16:9), la durée, et si possible une image de référence. Plus le brief est spécifique, plus l'agent produira un résultat exploitable. Évitez les objectifs vagues du type "fais quelque chose de sympa".

### Quel est le coût d'utilisation de Runway Agent Skills ?
Runway ne communique pas encore de tarification spécifique à Agent Skills. Les crédits sont consommés selon les générations effectuées par l'agent. Sur un brief complexe impliquant plusieurs clips et de la localisation, le coût peut vite dépasser celui d'une génération unique. Testez d'abord avec des briefs courts pour calibrer la consommation.
